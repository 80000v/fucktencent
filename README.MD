#反腾讯安全管家检测系统源码
![Alt text](http://ww1.sinaimg.cn/large/006fVPCvjw1f1rwp4h05bj309103adg3.jpg)
#####第二代反腾讯网址安全检测系统改用IP段拦截加header特征拦截，拦截率高达100%，同时有反举报的功能。经过一段时间的测试，新建的秒赞网站不会再被腾讯拉黑了。


使用方法，在程序的全局引用文件中引用这个php即可.
